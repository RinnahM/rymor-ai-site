Media & Content Creation — YouTube Comment Triage

AI Agent: Social Media Comment Responder

Live scenario canvas — Make.com

The Problem

Rins Media's YouTube channel maintains an Airtable base with a record for every published video — topic, promo codes, reference links. As the subscriber base grew, so did comment volume, and the two-person editorial team could no longer read and respond to every comment individually. Routine questions and higher-stakes complaints, refund requests, and business inquiries all landed in the same queue — with urgent messages sometimes missed entirely.

What I Built

An AI Agent scenario that watches for new YouTube comments, captures the video ID, and looks up that video's promo code and reference links in Airtable. The agent reads the comment against that context and classifies it as informational or requiring escalation. Informational requests (like "where's the discount code?") get a natural-language reply posted automatically; anything resembling a complaint, refund request, or business inquiry is routed straight to the team's Slack #support channel instead of being answered directly — so the automation never talks to an upset customer on the team's behalf.
